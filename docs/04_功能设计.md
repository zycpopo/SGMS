# 功能设计文档

## 程序架构

程序采用模块化设计，每个功能封装为独立的 `PROC`。主过程负责菜单分发：

```text
MAIN
 ├── LOAD_FILE → PARSE_CSV → PARSE_ONE_FIELD / PARSE_SCORE_FIELD
 ├── [菜单循环]
 │   ├── ADD_RECORD
 │   ├── SEARCH_RECORD
 │   ├── MODIFY_RECORD
 │   ├── DELETE_RECORD
 │   ├── SHOW_STATS → CALC_DIST
 │   ├── DRAW_HISTOGRAM → CALC_DIST → DRAW_ONE_BAR
 │   └── DISPLAY_ALL → DISP_ONE → PRINT_FLD
 └── SAVE_FILE → BUILD_CSV_LINE → COPY_TRIMMED
```

## 数据结构

### 记录格式（43 字节）

```asm
; 偏移   大小   字段
;   0     8    学号        (ASCII，右补空格)
;   8    16    姓名        (ASCII，右补空格)
;  24    16    课程        (ASCII，右补空格)
;  40     2    成绩        (二进制 word，0-100)
;  42     1    删除标记    (0 = 有效，非零 = 已删除)
```

### 关键变量

```asm
STUDENTS    DB MAX_STUDENT * RECORD_SIZE DUP(0)  ; 4300 字节
SCOUNT      DW 0                                  ; 总记录数（含已删除）
DIST        DW 5 DUP(0)                           ; 分布计数
ST_TOTAL    DW 0                                  ; 总分累加
ST_MAX      DW 0                                  ; 最高分
ST_MIN      DW 100                                ; 最低分
ST_CNT      DW 0                                  ; 有效记录数
```

## 文件操作

### 加载流程

```text
1. INT 21H AH=3DH (以只读方式打开 SCORE.TXT)
   ├── 成功 → AH=3FH (最多读取 4096 字节) → PARSE_CSV → AH=3EH (关闭)
   └── 失败 → AH=3CH (创建文件) → AH=3EH (关闭) → 以空数据启动
```

### CSV 解析算法

```text
SI = 文件缓冲区起始地址
WHILE SI < 缓冲区末尾 AND 记录数 < 100:
    跳过 CR/LF 字节
    DI = STUDENTS + 记录数 * 43

    解析学号:   复制最多 8 个字符或遇 ',' → 空格补齐至 8 字节
    解析姓名:   复制最多 16 个字符或遇 ',' → 空格补齐至 16 字节
    解析课程:   复制最多 16 个字符或遇 ',' → 空格补齐至 16 字节
    解析成绩:   累加十进制数字 → 二进制 word
    删除标记 = 0
    记录数++
```

### 保存流程

```text
1. INT 21H AH=3CH (创建/覆盖 SCORE.TXT)
2. FOR 每条记录 IN STUDENTS[0..SCOUNT-1]:
     IF 删除标记 == 0:
       BUILD_CSV_LINE → 去尾部空格，格式化为 "ID,Name,Course,Score\r\n"
       AH=40H (写入文件)
3. AH=3EH (关闭文件)
```

### 去空格算法（COPY_TRIMMED）

```text
输入:  SI = 字段起始, CX = 字段最大长度, DI = 目标缓冲, BX = 行长度
输出:  DI 前进, BX += 已复制字符数

1. 从字段末尾反向扫描找到最后一个非空格字符
2. 有效长度 = 最后一个非空格位置 + 1
3. 复制有效长度个字符从源到目标
4. BX += 有效长度
```

## CRUD 操作

### 添加记录

```text
1. IF SCOUNT >= 100 → 显示错误，返回
2. DI = STUDENTS + SCOUNT * 43
3. 提示输入学号 → 读取输入 → 空格补齐至 8 字节 → 存入 [DI]
4. 提示输入姓名 → 读取输入 → 空格补齐至 16 字节 → 存入 [DI+8]
5. 提示输入课程 → 读取输入 → 空格补齐至 16 字节 → 存入 [DI+24]
6. 提示输入成绩 → ATOI → 校验 0-100 → 存入 word 至 [DI+40]
7. [DI+42] = 0 (有效)
8. SCOUNT++
```

### 查找记录

```text
1. 提示选择查找方式: 1=按学号, 2=按姓名
2. 构建搜索关键字:
   - 按学号: 输入补空格至 8 字符
   - 按姓名: 输入补空格至 16 字符
3. FOR 每条记录 IN STUDENTS[0..SCOUNT-1]:
     IF 已删除 → 跳过
     用 REPE CMPSB 比较关键字与记录字段
     IF 匹配 → 显示记录，匹配计数+1
4. IF 匹配计数 == 0 → 显示 "未找到"
```

### 修改成绩

```text
1. 提示输入学号，构建搜索关键字
2. 遍历记录查找匹配:
     IF 找到 → 显示当前记录
     提示输入新成绩 → 校验 0-100 → 更新 [SI+40]
     显示成功消息
     RETURN
3. 未找到 → 显示 "未找到"
```

### 删除记录（逻辑删除）

```text
1. 提示输入学号，构建搜索关键字
2. 遍历记录查找匹配:
     IF 找到 → 显示当前记录
     [SI+42] = 1 (标记删除)
     显示成功消息
     RETURN
3. 未找到 → 显示 "未找到"
```

## 统计功能

### 计算算法

```text
初始化: total=0, max=0, min=100, count=0, dist[5]=0

FOR 每条记录 IN STUDENTS[0..SCOUNT-1]:
    IF 已删除 → 跳过
    score = [SI+40]
    total += score
    count++
    IF score > max → max = score
    IF score < min → min = score

    IF score >= 90     → dist[0]++
    ELSE IF score >= 80 → dist[1]++
    ELSE IF score >= 70 → dist[2]++
    ELSE IF score >= 60 → dist[3]++
    ELSE               → dist[4]++

average = total / count  (整数除法)
```

### 分数段划分

| 范围 | 标签 | 对应 DIST 索引 |
|------|------|----------------|
| 90-100 | 90-100 | DIST[0] |
| 80-89 | 80-89 | DIST[2] |
| 70-79 | 70-79 | DIST[4] |
| 60-69 | 60-69 | DIST[6] |
| <60 | <60 | DIST[8] |

## 关键算法

### ASCII 转整数（ATOI）

```text
输入:  SI = 字符串起始, CX = 字符串长度
输出:  AX = 整数值

result = 0
FOR 字符串中每个字符:
    IF char < '0' OR char > '9' → break
    result = result * 10 + (char - '0')
RETURN result
```

汇编实现：

```asm
ATOI PROC
    XOR  AX, AX           ; result = 0
AI_LP:
    MOV  BL, [SI]
    CMP  BL, '0'
    JB   AI_DONE
    CMP  BL, '9'
    JA   AI_DONE
    SUB  BL, '0'
    MOV  DX, 10
    MUL  DX               ; AX *= 10
    XOR  BH, BH
    ADD  AX, BX           ; AX += digit
    INC  SI
    LOOP AI_LP
AI_DONE:
    RET
ATOI ENDP
```

### 整数转 ASCII（ITOA_SCORE）

```text
输入:  AX = 整数 (0-100)
输出:  SCORESTR = 右对齐 ASCII，前导空格

1. 从 SCORESTR+5 开始，放置 '$' 结束符
2. 反复除以 10，余数 + '0' 反向存储
3. 剩余位置用空格填充
```

### 字符串比较（查找用）

```text
输入:  SI = 记录字段, DI = 搜索关键字, CX = 比较长度
输出:  ZF = 1 表示相等

使用 REPE CMPSB:
- 比较 CX 个字节
- 首次不匹配时停止 (ZF=0)，或完全匹配 (ZF=1)
```

## 输入处理

- 缓冲输入通过 INT 21H AH=0AH：最多 19 个字符
- 输入缓冲格式：`[最大长度][实际长度][字符...]`
- 菜单选择：单字符通过 INT 21H AH=01H
- 无效菜单输入：忽略，菜单重新显示
- 无效成绩输入：重新提示直到有效

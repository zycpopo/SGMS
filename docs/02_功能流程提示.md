# 功能流程提示

请按下面要求实现完整项目。

## 角色

你现在是一个熟悉 **MASM、8086、16 位 DOS 汇编、DOS int 21h** 的工程师。请生成一个真实可编译、真实可运行的 DOS 汇编程序。

## 项目名称

**学生课程成绩管理系统 (Student Grade Management System)**

## 最终目标

实现一个运行在低版本 DOS 环境下的学生成绩数据管理系统。程序必须使用 MASM 语法，文件名为：

```text
SCORE.ASM
```

编译方式：

```dos
MASM SCORE.ASM;
LINK SCORE.OBJ;
SCORE.EXE
```

## 绝对约束

必须遵守：

```asm
.MODEL SMALL
.STACK 100H
.DATA
.CODE
```

禁止使用 EAX/EBX/ECX/EDX、`.386`、`.MODEL FLAT`、32 位保护模式、Windows API、NASM 专用语法、Linux 系统调用。

程序界面文字必须使用英文 ASCII。源码注释可用中文。

## 功能要求

### 1. 文件加载（启动时）

程序启动时尝试以只读方式打开 `SCORE.TXT`（INT 21H AH=3DH）。如果文件存在，将全部内容读入 4096 字节缓冲区并解析 CSV。如果文件不存在，创建新文件（AH=3CH）并以空数据集启动。

**CSV 格式（每行）：**

```text
学号,姓名,课程,成绩
```

示例：
```
20240001,Alice,Calculus,85
20240002,Bob,English,92
```

**解析逻辑：** 按逗号分隔每行。学号（最多 8 字符）、姓名（最多 16 字符）、课程（最多 16 字符）在内存中右补空格。成绩解析为十进制→二进制字（0-100）。跳过空行和 CR/LF。

### 2. 文件保存（退出时）

选择退出（菜单选项 0）时，创建/覆盖 `SCORE.TXT`（AH=3CH），遍历所有未删除记录，去除每个文本字段的尾部空格，将成绩转回十进制 ASCII，将每条记录格式化为以 CR+LF 结尾的 CSV 行写入（AH=40H），然后关闭文件（AH=3EH）。

### 3. 添加记录

- 检查容量（最多 100 条）
- 依次提示输入：学号（1-8 位）、姓名、课程、成绩（0-100）
- 校验成绩范围
- 文本字段定长空格填充
- 存入 `STUDENTS + SCOUNT * RECORD_SIZE`
- SCOUNT 加 1

### 4. 查找记录

- 选择查找方式：1 = 按学号，2 = 按姓名
- 构建搜索关键字（学号补空格到 8 位，姓名补空格到 16 位）
- 遍历所有未删除记录，用 `REPE CMPSB` 比对
- 以表格格式显示匹配记录
- 无匹配时显示提示信息

### 5. 修改成绩

- 按学号查找
- 显示找到的记录
- 提示输入新成绩（校验 0-100）
- 更新内存中的成绩字段

### 6. 删除记录（逻辑删除）

- 按学号查找
- 显示找到的记录
- 将删除标记（偏移 42 处的字节）设为 1
- 物理数据保留在数组中；显示/查找/统计/保存时跳过

### 7. 统计

统计所有有效（未删除）记录：
- 有效记录数
- 平均分（总分 / 有效记录数，整数除法）
- 最高分
- 最低分
- 五段分数分布：90-100、80-89、70-79、60-69、<60

### 8. 柱状图

- 计算分布后，找出各段中最大人数
- 缩放：如果最大人数 > 50，每个 `#` = max/50 + 1 人；否则每个 `#` = 1 人
- 用 DOS INT 21H AH=02H 逐字符绘制 `#`
- 每段后面显示人数

### 9. 显示全部记录

- 显示表头：`ID / Name / Course / Score`
- 逐行显示每条未删除记录，列对齐

### 0. 保存并退出

- 调用保存子程序
- 显示再见信息
- 通过 `MOV AH, 4CH / INT 21H` 退出

## 数据结构

在 `.DATA` 段中：

```asm
MAX_STUDENT   EQU 100
RECORD_SIZE   EQU 43
ID_LEN        EQU 8
NAME_LEN      EQU 16
COURSE_LEN    EQU 16

STUDENTS    DB MAX_STUDENT * RECORD_SIZE DUP(0)
SCOUNT      DW 0
```

记录布局（43 字节）：
- 字节 0-7：学号（空格填充）
- 字节 8-23：姓名（空格填充）
- 字节 24-39：课程（空格填充）
- 字节 40-41：成绩（二进制 word）
- 字节 42：删除标记（0=有效，1=已删除）


## PRINT_STR 宏

定义宏以简化字符串输出：

```asm
PRINT_STR MACRO msg
    PUSH AX
    PUSH DX
    MOV  AH, 09H
    LEA  DX, msg
    INT  21H
    POP  DX
    POP  AX
ENDM
```

## 文件 I/O 参考

| 操作 | AH | 参数 |
|------|----|------|
| 创建文件 | 3CH | CX=0, DS:DX=文件名 |
| 打开文件（读） | 3DH | AL=0, DS:DX=文件名 |
| 读文件 | 3FH | BX=句柄, CX=字节数, DS:DX=缓冲区 |
| 写文件 | 40H | BX=句柄, CX=字节数, DS:DX=数据 |
| 关闭文件 | 3EH | BX=句柄 |

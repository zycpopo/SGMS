# 界面与布局设计规范

## 总原则

学生课程成绩管理系统采用**简洁文本模式界面**，设计目标是在标准 DOSBox 环境下无需彩色 BIOS 调用即可清晰显示。界面重点在于信息层次分明、间距统一、表格可读性高。

本项目有意不依赖彩色显示，确保：
- 与标准 DOSBox 完全兼容（无需额外配置）
- 与 FreeDOS 和真实 DOS 硬件兼容
- 输出在任何终端上清晰可读
- 重点放在数据管理逻辑而非视觉效果

如需添加颜色，可参考本文末尾的"可选颜色增强"部分。

## 菜单页设计

```text
+======================================+
|   Student Grade Management  v1.0    |
+======================================+
|  1. Add Record                       |
|  2. Search Record                    |
|  3. Modify Score                     |
|  4. Delete Record                    |
|  5. Statistics                       |
|  6. Histogram (score distribution)   |
|  7. List All Records                 |
|  0. Save & Exit                      |
+======================================+
Select (0-7):
```

**规则：**
- 边框使用 `+`、`=`、`|`、`-` 字符
- 标题在 40 字符宽度内居中
- 菜单项左对齐，缩进 2 个空格
- 提示符在底部边框之后

## 记录显示（表格）

```text
ID        Name             Course           Score
--------  ----------------  ----------------  -----
20240001  Alice             Calculus           85
20240002  Bob               English            92
20240003  Charlie           Calculus           78
```

**规则：**
- 表头行含列名
- 分隔线使用 `-`
- 列：ID（8 字符）、姓名（16 字符）、课程（16 字符）、成绩（右对齐，最多 3 位）
- 列间距 2 个空格
- 每条记录单独一行

## 柱状图页

```text
+========== Score Histogram ==========+
  (each # = 1 person(s))

  90-100: ### ppl
  80-89 : ##### ppl
  70-79 : #### ppl
  60-69 : ## ppl
  <60   : # ppl
```

**规则：**
- 标题使用 `+====...====+` 样式
- 比例说明解释柱状图含义
- 每段显示：范围标签、`#` 字符、人数
- 当某段最大人数超过 50 时自动缩放

## 状态消息

| 类型 | 前缀 | 示例 |
|------|------|------|
| 成功 | `[OK]` | `[OK] Record added.` |
| 信息 | `[INFO]` | `[INFO] No records.` |
| 错误 | `[ERROR]` | `[ERROR] Capacity full (100 max)!` |

## 输入提示

```text
Student ID (1-8 chars): _
Name: _
Course: _
Score (0-100): _
New score (0-100): _
```

## 不允许的界面

不要这样写：

```text
1.Add 2.Search 3.Modify 4.Delete 5.Stats 6.Hist 7.List 0.Exit Select:_
```

不要这样写：

```text
ID:20240001 Name:Alice Course:Calculus Score:85
```

必须有边框、间距、对齐列、清晰标签，每条信息单独占一行。

## 可选颜色增强

如果需要添加颜色，可增加基于 BIOS `int 10h AH=09h` 的彩色输出子程序：

```asm
; BL = 颜色属性（如 0Eh = 黄色，0Ah = 绿色）
PrintColorChar PROC
    PUSH AX
    PUSH BX
    PUSH CX
    MOV  AH, 09H
    MOV  BH, 00H
    MOV  CX, 1
    INT  10H
    ; BIOS 字符输出后需手动移动光标
    ...
    POP  CX
    POP  BX
    POP  AX
    RET
PrintColorChar ENDP
```

推荐颜色方案（如需添加）：

| 用途 | 属性 | 效果 |
|------|------|------|
| 边框 | 0Bh | 黑底亮青 |
| 标题 | 0Eh | 黑底亮黄 |
| 成功 | 0Ah | 黑底亮绿 |
| 错误 | 0Ch | 黑底亮红 |
| 普通文字 | 07h | 黑底浅灰 |
| 提示信息 | 0Fh | 黑底亮白 |
